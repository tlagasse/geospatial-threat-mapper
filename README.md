# geospatial-threat-mapper
Interactive web platform for visualizing and analyzing cyber threats using geospatial intelligence techniques

## Planned Features
- Interactive world map displaying cyber threats in real-time
- Integration with threat intelligence feeds (AbuseIPDB, AlienVault OTX)
- Geospatial pattern analysis and clustering
- Timeline-based threat tracking
- Threat actor attribution and tracking

## Tech Stack
- **Frontend**: React + Leaflet.js
- **Backend**: Python Flask
- **Database**: PostgreSQL with PostGIS
- **Data Sources**: AbuseIPDB, AlienVault OTX, MaxMind GeoLite2

## Background
This project combines my experience as a former military intelligence analyst (FMV/GEOINT) with cybersecurity, applying pattern-of-life analysis and geospatial techniques to threat intelligence.
